Job Analyzer

A web app created with Flask in Python that attempts to find keywords in a job posting. Users input the URL of the job posting they are interested in, then the app scrapes it, stores the data, calculates the TF-IDF, and then displays the results. TF-IDF (term frequency-inverse document frequency) is the algorithm the app uses to find keywords. A web scraper made in python with the Beautiful Soup library compiled the initial corpus of data used to train the algorithm. MongoDB was used to store all data.

DNS Resolver

Used Java to create a program that would decode a user’s domain name request, and save the request. Then send the request to google for an answer. It would then cache google’s answer for future requests and formulate a response packet and send that back to the user.

Sharks and Minnows

A team project to make a game of tag coded in c++. A user class tracks the mouse’s position and continuously redraws the user sprite in that position. An enemy class tracks the position of the mouse and updates its position with increased speed to move closer to the mouse. The game ends when a shark sprite touches the user sprite and an end screen displays how long you survived.

MSDscript

A baseline scripting language programmed in C++ capable of interpreting and solving simple expressions. It can handle Addition, Multiplication, Comparison, if/then expressions, let expressions (e.g. (x = 5 in (x + 5))), and functions (e.g. (f(x) = x+5)). This project includes automated and randomly generated tests. It also includes extensive unit testing.
